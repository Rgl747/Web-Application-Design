## Descripción de Markdown

---

Markdown es un lenguaje de marcado ligero para crear texto formateado utilizando un editor de texto sin formato.

| Ejemplo  | Salida  |
| --- | --- |
| # Titulo | Titulo |
| ## Subtitulo | Subtitulo |
| ** Negritas ** | **Negritas** |
| [Link]-() | Imagen del link |
